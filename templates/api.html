API Endpoint
            
https://apinim.herokuapp.com/api/

The Westeros API provides programmatic access to read Game of Thrones data. Retrieve a character, provide an oauth connexion, retrieve a familly, filter them, etc.

To use this API, you need an API key. Please contact us at jon.snow@nightswatch.wes to get your own API key.

get information

To get result you need to make a GET call to the following url :
https://apinim.herokuapp.com/api/anime


Python 

import requests
result = requests.get('https://apinim.herokuapp.com/api/anime?q=Kyoukai+suiri')
print(result.json())

JS 

import axios from "axios";

const options = {
    method: 'GET',
    url: 'https://apinim.herokuapp.com/api/anime',
    params: {q: 'Kyoukai suiri'},
    headers: {
        'Access-Control-Allow-Origin': '*'
    }
};

axios.request(options).then(function (response) {
    console.log(response.data);
}).catch(function (error) {
    console.error(error);
});

PHP 

                    
$request = new HttpRequest();
$request->setUrl('https://apinim.herokuapp.com/api/anime');
$request->setMethod(HTTP_METH_GET);

$request->setQueryData([
'q' => 'Maoujo de oyasumi'
]);

try {
    $response = $request->send();
    
    echo $response->getBody();
} catch (HttpException $ex) {
    echo $ex;
}
