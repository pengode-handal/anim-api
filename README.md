# Welcome
Thanks for using this repo


## Requirements
- python3
- pip

## Installation
```bash
git clone https://github.com/pengode-handal/anim-api
cd anim-api
pip3 install -r requirements.txt
```

## Running on server
```bash
python3 app.py
```
